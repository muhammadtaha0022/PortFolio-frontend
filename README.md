## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/muhammadtaha0022/PortFolio-frontend.git

2. **Navigate to project directory**
   ```
   cd PortFolio-frontend
   ```
3. Open in browser

   * Double-click index.html or

   * Use VS Code Live Server extension
